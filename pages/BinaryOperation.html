Вступ

Так як комп’ютер вміє оперувати тільки 0 та 1, це відкриває можливість для деяких операцій, які не існують в десятковій системі числення, бо можна вважати, що 1 – правда і 0 – хиба. І тоді в нас є 6 операції:

  • Побітове АБО
  • Побітове І
  • Побітове ВИКЛЮЧНЕ АБО
  • Побітове НІ
  • Побітове ЗСУВ ВПРАВО
  • Побітове ЗСУВ ВЛІВО

Ці операції працюють побітово, тобто операції виконують дію з парою бітів або одним бітом. Тобто операції над однією парою бітів не впливають на результат наступної пари бітів.

Побітовий АБО (BIT OR)

Ця побітова операція дає 1 в результат, якщо хоча б один з бітів пари рівний 1.

A 0 1 0 1
B 0 0 1 1
Result 0 1 1 1

Нижче ви можете переглянути, як відбувається побітове АБО з 2 числами.


Побітовий І (BIT AND)

Побітову І дає 1 в результат тільки тоді, коли обидва біти в парі рівні 1.

A 0 1 0 1
B 0 0 1 1
Result 0 0 0 1

Нижче ви можете переглянути, як відбувається побітове І з 2 числами.


Побітовий ВИКЛЮЧНЕ АБО (BIT XOR)

Ця операція дає 1 в результат тільки тоді, коли рівно один біт з пари рівний 1, а інший рівний 0.

A 0 1 0 1
B 0 0 1 1
Result 0 1 1 0

Нижче ви можете переглянути, як відбувається побітове ВИКЛЮЧНЕ АБО з 2 числами.


Побітовий НІ (BIT NOT)

Перш за все, ця операція працює тільки з одним числом.

Побітове НІ інвертує біт, який воно зараз опрацьовує.

A 0 1
Result 1 0

Побітовий ЗСУВ (BIT SHIFT)

Побітовий зсув буває 2 видів: побітовий зсув вправо та побітовий зсув вліво. Ці дві операції працюють тільки над одним числом. Для реалізації цього зсуву потрібні два числа: те, яке змінюємо (вхідне) та те, яке вказує на скільки бітів робимо зсув.
Побітовий зсув вправо та вліво зсуває біти числа, на якусь кількість позицій вправо та вліво відповідно.

Нижче ви можете переглянути, як відбувається побітове ПОБІТОВИЙ ЗСУВ, і вибрати його напрямок у випадаючому списку.
